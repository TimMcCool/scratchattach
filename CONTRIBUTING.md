# Contributing

You can contribute to scratchattach using the available GitHub features.

## LLMs

When opening a pull request or issue, you must disclose any use of LLMs (AI) in your written content.
This disclosure should include what the LLM wrote and the extent to which it was used.
If we receive a pull request or issue that does not follow this rule, as determined by us, it will be closed.
If you repeatedly disregard this rule, we will take the necessary actions to ensure compliance or prohibit you from making further contributions.
